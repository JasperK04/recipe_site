About Recipe Site

Recipe Site is your personal cookbook in the cloud.

Features

  • Add and organize your favorite recipes
  • Edit and update recipes anytime
  • Browse recipes by category
  • Search for specific dishes
  • Track prep and cook times
  • Secure user authentication

Getting Started

  1. Create an account
  2. Log in to your account
  3. Add your first recipe
  4. Start building your personal cookbook!
Technology Stack
  • Flask - Web Framework
  • SQLAlchemy - Database ORM
  • Flask-Login - Authentication
  • Flask-Session - Session Management
  • Bootstrap 5 - UI Framework
